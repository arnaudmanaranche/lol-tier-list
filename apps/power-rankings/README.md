## Dev mode

Launch the app by running the following command

    $ yarn dev

## Build the app

Build the app by running the following command

    $ yarn build

## Start the app

Start the app by running the following command

    $ yarn start

## Run tests

Launch tests suits by running one of the following command

    $ yarn test
    $ yarn test:watch
